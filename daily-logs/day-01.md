## 🌐 What is the Web?
- A system of interconnected documents, media files, and resources
- It is a **subset** of the broader Internet (Internet > Web)
- Users can access it via the internet

---

## 💻 What is Web Development?
The process of creating websites and web applications

| | Website | Web Application |
|---|---|---|
| Content | Static, Read-only | Dynamic |
| User Interaction | Minimal | High |
| Examples | Wikipedia, Blogs | LinkedIn, Instagram, Facebook |

---

## 🏗️ 3 Layers of a Web Application

### 1. 🎨 Frontend / UI (User Interface)
- The visual layer that users **see and interact with**
- Includes design, buttons, text, and graphics
- Technologies: **HTML, CSS, JavaScript, React**

### 2. 🧠 Backend
- The **"brain"** of the application
- Handles business logic, processing, and rules
- Examples: verifying login, calculating game winner
- Technology: **Node.js**

### 3. 🗄️ Database (DB) Layer
- Responsible for **storing and retrieving** user data
- Types: SQL, NoSQL

---
